 Comandos do jogo:
 -----------------
 São usados 3 padroes para o jogo. Um para o tabuleiro e 
 os outros dois para controlar o declive em x e em y do tabuleiro.
 Para reiniciar o jogo usar a tecla "R" e para mudar entre nivel do jogo usar "T".



 Diretorios do zip:
 ------------------
 ..\RVAu1_ARToolkit\include (ficheiros de inclusão do ARToolkit)

 ..\RVAu1_ARToolkit\lib (bibliotecas do ARToolkit)

 ..\RVAu1_ARToolkit\patterns (padroes do ARToolkit)
 


 Projecto VS2010:

 ----------------

 ..\RVAu1_ARToolkit\RVAu1\RVAu1.sln

 

 Executavel da aplicação:

 ------------------------

 ..\RVAu1_ARToolkit\RVAu1\Release\RVAu1.exe

 ou

 ..\RVAu1_ARToolkit\RVAu1\Debug\RVAu1.exe
